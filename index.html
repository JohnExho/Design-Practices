Quamet Attendance Monitoring System

Overview

Quamet is a Card-based Human Monitoring Function (CHMF) designed for efficient and accurate attendance tracking using RFID technology.

Features

  • ✅ RFID-based check-in and check-out
  • ✅ Real-time attendance tracking
  • ✅ Secure user authentication
  • ✅ Role-based access control (Admin, Teacher)
  • ✅ Auto-generated reports and logs

Technology Stack

Backend: Laravel (PHP)
Frontend: Blade, Bootstrap, JavaScript
Database: MySQL
Authentication: Laravel Auth
